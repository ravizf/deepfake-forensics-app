Checkpoint Setup

  • Option 1: place deepfake_model.pth in the project root beside app.py, forensics.py, and model_manifest.json.
  • Option 2: if the model is in another folder, update checkpoint_path in model_manifest.json to the correct relative path such as models/deepfake_model.pth.
  • Restart the server after adding the file.
  • Look for MODEL LOADED: True in the terminal.
  • When loading succeeds, /api/analyze should switch from fallback_demo to trained_model and stop using Heuristic fallback.
